## Development

We use `node` version `14.15.1`

```
nvm install 14.15.1
```

```
nvm use 14.15.1
```

The first time, you will need to run

```
npm install
```

Then just start the server with 

```
npm run start
```
It uses nodemon for livereloading :peace-fingers:

---

To run tests use

```
npm run test
```

To build the app use

```
npm run build
```
